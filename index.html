MacPadPlusPlus Feature Overview

MacPadPlusPlus is a Notepad++-inspired editor for macOS with fast large-file handling, search workflows, navigation tools, and customizable highlighting.

What the App Can Do

This page mirrors the built-in help guide and shows the main feature areas with real UI captures.

Interface Overview

  • Toolbar actions for common commands
  • One file per tab with quick close controls
  • Status bar with file and cursor metadata

Editing Basics

  • Undo/redo and clipboard operations
  • Line numbers and horizontal scrolling
  • Live caret and line position feedback

Find Dialog

  • Search history and quick actions
  • Case, whole-word, and wrap options
  • Search modes similar to Notepad++

Replace Dialog

  • Replace one, replace all, and feedback state
  • Integrated options for controlled edits
  • Consistent workflow with Find dialog

Search Results Panel

  • Grouped results by search and by file
  • Fast navigation to result lines
  • Context menu copy actions for result sets

Marks and Bookmarks

  • Bookmark matching lines from Mark tab
  • Gutter indicators and status-bar counts
  • Useful for repeated log scanning

Token Styling

  • Style all occurrences from context menu
  • Color-code important tokens instantly
  • Clear styles when investigation is complete

Function List

  • Detects symbols based on language rules
  • Filter list and jump directly to targets
  • XML and regex-driven rule support

User Defined Languages (UDL)

  • Import UDL XML definitions
  • Apply custom syntax highlighting rules
  • Use custom formats from Language menu

Additional Capabilities

  • Line operations: duplicate, move, split/join, sort, remove empties, and beautify JSON/XML.
  • Macros: start/stop recording, playback, run multiple times, or run to end-of-file.
  • View controls: word wrap, zoom, and right-panel symbol navigation with Function List.
  • Language modes: automatic by extension, manual override, or plain text mode.
